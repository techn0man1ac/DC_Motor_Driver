# BJT/MOSFET DC Motor Driver

- Vin 10..20 V;
- PWM(EN pin, active LOW level) up to 1 Khz(with PC817 optocoupler);
- Current 2..4 A(Q1, Q4 - TIP122 / Q2, Q3 - TIP127, FU1 5A), 3..9 A(Q10, Q6 - IRF9540 / Q9, Q5 -IRF540, FU1 10A).
